# ocean.js demo

## Setup
Run
```
npm i
```

- Set a private key and export it
```
export MNEMONIC="XXXX"
```
- Set a NETWORK
````
export NETWORK='rinkeby'
````
- Set an RPC
````
export RPC='XXXX'
````


## Quick  start
- Run the following commands
````
# Get help
npm run cli h

````
- Publish an asset
````
npm run cli publish

````
